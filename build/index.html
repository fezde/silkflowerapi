Silk Flower API

Intro

Welcome to the Silk Flower API. If you need a nice random silk flower for your site or project, feel free to use this API.

This API provides a new, randomly created Silk Flower about every 10 minutes.

The images created by this API are licensed under a Creative Commons Attribution-ShareAlike 4.0 International License.

Example

By including something like this <img src="https://fezde.github.io/silkflowerapi/flower.jpg" width="256" alt="Silk Flower Example" /> into your page, you will include an image like this:

Or if you want to include it into your project, just use this URL in the according request:
https://fezde.github.io/silkflowerapi/flower.jpg
